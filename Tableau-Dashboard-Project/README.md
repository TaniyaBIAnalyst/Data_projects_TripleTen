# Return Rate Analysis Dashboard

## Project Description
This project analyzes product return trends using an interactive Tableau dashboard. The dashboard explores how return rates vary across product categories, sub-categories, time periods, geographic regions, and customers. The goal of the project was to identify areas with higher return activity and provide business recommendations to help reduce returns and improve overall performance.

## Business Problem
High return rates can reduce profit, create inventory challenges, and lower customer satisfaction. This project helps the business understand where returns are happening most often and what patterns may be contributing to increased return activity.

## Tools Used
- Tableau Public
- Data visualization
- Dashboard design
- Charts and filters
- Geographic analysis
- Business analysis
- Data storytelling

## Dashboard Features
- Return rate analysis by product category
- Sales and return comparison by sub-category
- Average return rate over time
- Return rate analysis by state
- Customer-level return behavior
- Final summary with business recommendations

## Key Insights
- Return rates vary across product categories, with some categories showing higher average return rates than others.
- Products with higher sales volumes also tend to show noticeable return activity, especially across certain sub-categories.
- Return rates fluctuate over time, with some months showing clear spikes.
- Return behavior differs by state, suggesting that geographic factors may influence return patterns.
- Most customers have similar return behavior, but a few customers show noticeably higher return activity.

## Business Recommendations
- Focus on improving product quality and product descriptions for high-return sub-categories.
- Monitor peak sales periods closely because return spikes may happen during or after high-sales months.
- Review regional return trends to identify states or markets that may need different product, shipping, or customer communication strategies.
- Investigate customers or segments with unusually high return activity to understand possible causes.
- Use the dashboard regularly to track return trends and support data-driven decisions.

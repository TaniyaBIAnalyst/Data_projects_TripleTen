# Manhattan Airbnb Analysis

## Project Description
This project analyzes Airbnb listing data in Manhattan to identify top-performing neighborhoods, pricing trends, and revenue opportunities. The goal was to use spreadsheet analysis to understand which areas and listing types showed the strongest business potential.

## Tools Used
- Google Sheets
- Spreadsheet formulas
- Pivot tables
- Charts
- Data cleaning
- Data analysis

## Project Files
- [View the Google Sheets project](https://docs.google.com/spreadsheets/d/1rhcYKBz-4Zu6ejDFe8uOJ8yAwAOKX1Lo3zkjWzwekS4/edit?usp=drive_link)

## Key Insights
- Identified top-performing Airbnb neighborhoods in Manhattan.
- Analyzed pricing trends across different neighborhoods and listing types.
- Found revenue opportunities based on listing patterns.

## Future Improvements
In the future, I would add more visualizations, compare seasonal trends, and build an interactive dashboard to make the insights easier to explore.
